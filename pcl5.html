PCL-5 Assessment

Below is a list of problems that people sometimes have in response to a very stressful experience. Please read each problem carefully and select one of the numbers to indicate how much you have been bothered by that problem in the past month.

0 = Not at all

1 = A little bit

2 = Moderately

3 = Quite a bit

4 = Extremely
